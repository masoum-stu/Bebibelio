
# AI Bible App

A single‑file offline Bible app with:

- Songs, Videos, Stories
- Children stories
- Bible navigation (Book → Chapter → Verses)
- AI Assistant demo with subscription popup
- Fully offline support
- GitHub Pages ready

## How to Use

Upload **index.html** and **README.md** to your GitHub repo root.

Then enable GitHub Pages:

**Settings → Pages → Branch: main → Folder: root → Save**

Your app will appear at:

```
https://USERNAME.github.io/REPO/
```
